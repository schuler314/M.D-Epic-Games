How to Play

💻 Controls (Desktop)

  • Left Click: Place block
  • Right Click: Delete block
  • R Key: Rotate Block / Thing
  • Shift + Drag: Speed Build / Speed Destroy
  • Drag / Scroll: Rotate camera & Zoom

📱 Controls (Touch / Tablet)

  • Tap: Place block at cursor.
  • Delete Mode: Tap the 🧨 Delete button, then tap blocks to remove them.
  • Paint Mode: Tap the 🖌️ Paint button to drag and build/destroy quickly (disables camera rotation).
  • Rotate: Tap the 'R / Tap' rotation button in the bottom toolbar.
  • Drag (1 Finger): Rotate camera.
  • Pinch (2 Fingers): Zoom in & out.

🎮 Controls (Gamepad 1P)

  • Left Stick: Move Cursor (N/S/E/W)
  • D-Pad Up/Down: Move Cursor Up/Down
  • Right Stick: Orbit Camera
  • A / Cross: Place Block
  • B / Circle: Delete Block
  • X / Square: Rotate Block
  • Shoulders (LB/RB): Next/Prev Material

👥 2-Player Controls (Player 2 Keyboard)

  • W,A,S,D Move Cursor | Q,E Up/Down
  • Space Place | X Delete | U Rotate
  • Hold Space/X + Move: Speed Build / Destroy

🚶 Walk Mode (First Person)

Experience your build! Use WASD to walk, Space to jump, and Mouse to look around. Stairs are climbed automatically. Left-Click on doors and windows to interact with them! Press ESC to exit.

💾 Save & Load

Use the Save button to download a JSON file of your world to your computer. Use Load to restore it.

The Catalog (100 Items)

Select an item to equip it. Press R or U to rotate it before placing.
