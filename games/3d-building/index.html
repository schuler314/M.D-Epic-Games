How to Play

🖱️ Controls

  • Left Click: Place block
  • Right Click: Delete block
  • Shift + Drag: Speed Build / Speed Destroy
  • Drag: Rotate camera
  • Scroll: Zoom in/out

🎮 Game Modes

  • Free Build: Build whatever you want! Watch the sky for falling powerups.
  • Blueprint: Race against the clock to fill in the ghost blocks and complete the house.

👥 2-Player Mode (Mouse vs Keyboard)

2-Player mode splits the screen so you can build together simultaneously!

Player 1 (Left Screen): Uses the Mouse.
Player 2 (Right Screen): Uses the Keyboard.
- W,A,S,D: Move Cursor   Q / E: Cursor Up/Down
- Space: Place Block   X: Delete Block
- Hold Space/X + Move: Speed Build / Speed Destroy
- Arrows: Orbit & Zoom Camera
- 1-6: Materials   7-9: Shapes   0,-,=: Things
